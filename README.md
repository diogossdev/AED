# 📚 Algoritmos e Estruturas de Dados (AED) - IFPB

Este repositório contém as listas de exercícios, desafios e projetos desenvolvidos na disciplina de Algoritmos e Estruturas de Dados do curso de Engenharia de Software do Instituto Federal da Paraíba (IFPB).

## 🛠️ Tecnologias Utilizadas
* Python 3
* Lógica de Programação

## 🗂️ Conteúdos Abordados
* Listas e Vetores
* Pilhas e Filas
* Árvores
* Grafos

## 💡 Sobre
O objetivo deste repositório é documentar minha evolução na resolução de problemas complexos e no entendimento de como os dados são estruturados e manipulados na memória.
